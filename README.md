# Node version 18.5.0

# Запуск приложения

## Сначала необходимо запустить json server

### `json-server db.json -m ./node_modules/json-server-auth`

Сервер запустится по адресу http://localhost:3000

## Далее создаем новый терминал и запускаем само приложение

С помощью 

### `yarn start`

или

### `npm start`
